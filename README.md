# Risk Master App
**Back End Server**

## Project setup
```
pip install -r requirements.txt
```

### Compiles and hot-reloads for development
1. Copy settings.ini file
2. Run the below commands:
    ```
    py manage.py makemigrations
    py manage.py migrate
    py manage.py runserver
    ```
